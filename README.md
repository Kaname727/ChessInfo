# おしゃれ時計＆メモ（Chrome拡張）

サイトの右上に、おしゃれな日付・時刻と任意のメモを表示するChrome拡張機能です。

## 機能

- **日付** … 例: `2025/2/11 (火)`
- **時刻** … 例: `14:32:05`（1秒ごとに更新）
- **任意の文字** … ポップアップで設定したメモを同じ枠内に表示

## インストール方法

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダ（`kikaku`）を選択

## 使い方

- どのサイトでも右上に時計パネルが表示されます
- 拡張アイコン（パズルピース → ピン留めでツールバーに表示）をクリックしてポップアップを開き、「表示する文字」を入力して「保存」を押すと、その文字が時計の下に表示されます
- メモを空にして保存すると、メモ部分は非表示になります

## ファイル構成

- `manifest.json` … 拡張の設定
- `content.js` … ページに時計・メモを表示するスクリプト
- `content.css` … 時計パネルの見た目
- `popup.html` / `popup.js` … メモを設定するポップアップ
