# Stylish Clock Memo

ブラウザ表示を邪魔しないミニマルなデザインで、日付・時刻・メモをページ右上に表示する Chrome 拡張機能です。  
作業中のメモや、サイト閲覧時に常に見ておきたい短いテキストを、シンプルに管理できます。

---

## Overview

この拡張機能は、閲覧中のページ上に固定パネルを表示し、次の情報をリアルタイムに確認できます。

- 日付（曜日つき）
- 時刻（1 秒ごとに自動更新）
- 任意のメモ（ポップアップで編集・保存）

情報は 1 つのコンパクトな UI にまとめられており、視認性と控えめな存在感のバランスを重視しています。

---

## Features

| 項目 | 内容 |
| --- | --- |
| Date | `YYYY/M/D (曜)` 形式で表示 |
| Time | `HH:MM:SS` を 1 秒ごとに更新 |
| Memo | ポップアップから自由入力。空欄保存で非表示 |

---

## Installation

1. Chrome で `chrome://extensions/` を開く
2. 右上の **デベロッパーモード** を有効化
3. **パッケージ化されていない拡張機能を読み込む** をクリック
4. このリポジトリのフォルダを選択

---

## Usage

1. 任意の Web サイトを開く
2. ページ右上に時計パネルが表示されることを確認
3. 拡張機能アイコンからポップアップを開く
4. 「表示する文字」にメモを入力して保存
5. 保存した内容が時計の下に反映される

メモを空欄で保存すると、メモ表示のみ自動で隠れます。

---

## Project Structure

- `manifest.json` : 拡張機能の定義
- `content.js` : 時計・日付・メモをページに描画
- `content.css` : パネルのスタイル
- `popup.html` : 設定ポップアップ UI
- `popup.js` : メモ保存処理

---

## Notes

- 本拡張はコンテンツスクリプトとして動作し、対応ページ上に UI を重ねて表示します
- サイト側のレイアウトや z-index によっては重なり順が変わる場合があります
